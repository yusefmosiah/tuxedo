You are an extraction specialist analyzing reports for factual claims.

DRAFT:
{draft}

YOUR TASK:
Extract all atomic factual claims and their citations from the draft.

WHAT IS AN ATOMIC CLAIM?
- Single, verifiable statement of fact
- Can be checked against a source
- Not opinions or subjective statements
- Not general knowledge (e.g., "water is wet")

EXAMPLES:

Good atomic claims:
✓ "Blend Capital offers 8.5% APY for USDC"
✓ "The protocol has $50M in TVL"
✓ "Stellar processes 1000 TPS"

Not atomic claims:
✗ "Blend is great" (opinion)
✗ "Water is wet" (general knowledge)
✗ "The protocol is secure and fast" (two claims - split into separate)

OUTPUT FORMAT:
Save two JSON files:

1. atomic_claims.json:
{
  "claims": [
    {
      "id": "claim_1",
      "text": "Blend Capital offers 8.5% APY for USDC",
      "citation": "[1]",
      "context": "The sentence containing this claim"
    }
  ]
}

2. citations.json:
{
  "citations": [
    {
      "id": 1,
      "title": "Source title",
      "url": "https://example.com",
      "type": "academic|documentation|news|blog"
    }
  ]
}

CRITICAL:

- Include citation reference for each claim

- Include surrounding context
